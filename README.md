# Employee-Equality-Analysis
Employee Equality Analysis Dashboard

# Project Objective

The objective of this project is to analyze employee equality scores across factories and machines using Tableau. The dashboard helps identify compensation fairness and highlights areas that require further investigation.

# Tools Used
Tableau,
Microsoft Excel

# Dashboard
Screenshot is added.

# Dashboard Features
Equality distribution by Factory
Equality distribution by Machine
Interactive Filters
Fair vs Unfair Classification

# Key Insights
The dashboard categorizes equality scores into Fair and Unfair classifications.
Factory-level analysis highlights locations with higher numbers of unfair equality scores.
Machine-level analysis helps identify operational areas that may contribute to compensation inequality.
Interactive visualizations make it easy to compare equality trends across different dimensions.

# Business Recommendations
Review compensation policies in factories with a higher number of unfair classifications.
Investigate machine-specific patterns to identify operational factors contributing to inequality.
Conduct regular equality audits to monitor compensation fairness.
Use dashboard insights to support data-driven HR decisions and improve workplace equity.
